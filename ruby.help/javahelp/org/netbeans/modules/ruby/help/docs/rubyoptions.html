Options Window: Ruby

Formatting tab

The Formatting tab enables you to set the formatting of code in the editor. Formatting options include setting the indentation size, reformatting comments, and indentation of HTML within RHTML and ERB blocks.

Hints tab

The Hints tab enables you to select which hints are displayed.

Miscellaneous tab

In the Miscellaneous tab, you can select Show Logical Project View. This selection enables you to toggle between a file view of Rails projects and a logical view in the Projects window, after restarting the IDE. The default project view is logical. You can also see a file view of your projects by choosing Window > Files.
