Options Window: Ruby

The Ruby Options window enables you to set the formatting of code in the editor and select which hints are displayed.

In the Miscellaneous tab, you can select Show Logical Project View. Toggle between a file view of Rails projects or a logical view in the Projects window, after restarting the IDE. The default view is logical. You can also see a file view of your projects by choosing Window > Files.
