Plugin Installation Settings Dialog Box

See Also 

Use this dialog box to indicate installation settings for a new Ruby on Rails plugin. It opens when you choose an available plugin in the Plugin Manager's New Plugins tab and click Install. Typically, you just click OK to install a plugin when this dialog opens.

  • When you click OK, an Installation dialog box opens and shows you the progress of your download. In addition, you might be prompted by the plugin's download program to specify some information for the plugin, so keep an eye on the Installation dialog while the download is happening.

You can set the following options:

  • Name. Specifies the name of the plugin. If you want to install a different plugin, it's easier to close the dialog box, select another plugin in the New plugins tab, and click Install.
