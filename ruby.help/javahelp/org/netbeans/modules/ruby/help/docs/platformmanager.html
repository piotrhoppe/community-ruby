Platform Manager

Use the Ruby Platform Manager to register Ruby platforms with the IDE. When you create a new Ruby project, you must associate the project with one of the registered platforms. You can use the project's Properties manager to change the project's platform to a different one on the list. To open the Ruby Platform Manager, choose Tools > Ruby Platforms from the main menu.

When you select a Ruby interpreter from the list, the dialog box displays the following information about the interpreter:

  • Platform Name. Indicates the name and version of the platform you are using.
  • Interpreter. Indicates the full path of the selected interpreter.
  • Gem Home. Indicates the path and folder where the main gem repository is located.
  • Gem Path. Specifies the location of the gem, which is required as you may have multiple gem repositories. To change the path, click the Browse button. Click the Add button to add a new gem path. Click the Remove button to delete the gem path that is currently displayed.
  • Gem Tool. Indicates the path to the autodetected gem tool.
  • Ruby Debugger. Indicates the default debugger.
  • Add Platform and Autodetect Platforms. To use a Ruby interpreter that is not on the list, click Add Platform. Click Autodetect Platforms to have NetBeans automatically detect the platforms that you have installed.
