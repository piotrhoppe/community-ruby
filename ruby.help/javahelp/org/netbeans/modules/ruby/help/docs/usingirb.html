Using Interactive Ruby (IRB)

See Also 

To open an interactive Ruby (IRB) shell in the IDE, choose Window > Other > Ruby Shell (IRB) from the main menu. The IRB shell opens in a tabbed window at the bottom of the IDE in the same location as the Output window. The IDE looks at Ruby Platform setting for the current project to determine which interpreter to use. The window's tab shows which interpreter is used.
