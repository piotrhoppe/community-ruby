Ruby or Rails Project Properties: Java Options

See Also

Use this properties dialog box to specify how your Ruby or Ruby on rails project uses Java. You open the dialog box by right-clicking the main project node in the Properties window, choosing Properties, and then clicking the Java node in the Project Properties dialog box.

  • These properties have meaning only if your interpreter is JRuby.
  • Include Java. If your project makes calls to Java libraries, you must include Java with the project.
  • Java Platform.
  • JRuby Class Path: Compile Time Libraries.
  • Build Projects on Classpath.
