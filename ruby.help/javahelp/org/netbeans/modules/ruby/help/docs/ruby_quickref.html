NetBeans Ruby Tasks: Quick Reference

See Also | Tutorials

This topic describes common tasks that you can perform with NetBeans Ruby. For more detailed information, click the links in the right column.

To perform this task... Follow these steps
Create a new Ruby or Ruby on Rails project
  1. Choose File > New Project from the main menu to open the New Project wizard.
  2. Select Ruby in the Categories list.
  3. In the projects list, select either New Ruby Project or New Ruby on Rails Project.
    • If you choose New Ruby Project, see New Ruby Application Wizard: Name and Location.
    • If you choose New Ruby on Rails Project, see New Ruby on Rails Application Wizard: Name and Location.
Import an existing Ruby or Ruby on Rails application
  1. Choose File > New Project from the main menu to open the New Project wizard.
  2. Select Ruby in the Categories list.
  3. In the projects list, select either Ruby Application with Existing Sources or Ruby on Rails Application with Existing Sources.
    • If you choose Ruby Application with Existing Sources, see New Ruby Project with Existing Sources Wizard: Name and Location.
    • If you choose Ruby on Rails Application with Existing Source, see New Ruby on Rails Project with Existing Sources Wizard: Name and Location.
Use the interactive Ruby shell (irb) in NetBeans
  • Choose Window > Other > Ruby Shell (IRB). For more information, see Using Interactive Ruby (IRB).
Choose a database server for a Ruby on Rails project
  • When you create a Ruby on Rails project (see above) and use the JRuby interpreter, you can use the New Ruby on Rails Application Wizard: Name and Location. panel to set the database server.

    For more information, see Using Database Servers with JRuby.
Register a database server
  • See "Registering a Database" in Using Database Servers with JRuby.
Add a database to the IDE
  • For instructions on adding some common databases, see "Adding Databases" in    
Manage your Ruby gems
  • Choose Tools > Ruby Gems to open the Ruby Gems Manager.
